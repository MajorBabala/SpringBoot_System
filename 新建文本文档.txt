基于 SpringBoot 的企业财务管理系统项目设计文档
1. 引言
1.1 项目背景
随着中小企业数字化转型的推进，传统手工财务管理方式已无法满足高效、准确、规范的管理需求。商业化财务软件成本高昂、功能冗余，难以适配中小企业个性化需求。本项目旨在开发一套轻量级、易用、安全的企业财务管理系统，帮助中小企业实现财务业务的线上化、规范化管理。
1.2 项目目标
•	实现凭证管理、账簿查询、报表生成、费用报销等核心财务功能。
•	提供基于角色的权限控制，确保数据安全。
•	采用前后端分离架构，保证系统的可扩展性和维护性。
•	系统响应快速，支持50+用户并发访问。
1.3 项目范围
本项目聚焦中小企业日常财务核算，不包括固定资产、工资、税务申报等模块，但预留接口供后续扩展。

3.2 前后端分离架构
•	前端通过 Axios 发送 HTTP 请求，后端返回 JSON 格式数据。
•	用户登录后，后端生成 JWT 令牌返回前端，前端存储于 localStorage，并在后续请求的 Authorization 头中携带。
•	后端通过 Spring Security 过滤器解析 JWT，完成认证和授权。

7.2.1 凭证填制与借贷平衡校验
伪代码：
text
public Result addVoucher(VoucherDto dto) {
    // 1. 参数校验
    if (dto.getEntries() == null || dto.getEntries().isEmpty()) {
        return error("分录不能为空");
    }
    
    // 2. 计算借方合计与贷方合计
    BigDecimal totalDebit = dto.getEntries().stream()
            .map(VoucherEntryDto::getDebitAmount)
            .reduce(BigDecimal.ZERO, BigDecimal::add);
    BigDecimal totalCredit = dto.getEntries().stream()
            .map(VoucherEntryDto::getCreditAmount)
            .reduce(BigDecimal.ZERO, BigDecimal::add);
    
    // 3. 借贷平衡校验
    if (totalDebit.compareTo(totalCredit) != 0) {
        return error("借贷金额不平衡");
    }
    
    // 4. 保存凭证（状态为未审核）
    Voucher voucher = convertToEntity(dto);
    voucher.setStatus(0);
    voucher.setMakerId(currentUserId());
    voucher.setCreateTime(new Date());
    voucherMapper.insert(voucher);
    
    // 5. 保存分录
    for (VoucherEntryDto entryDto : dto.getEntries()) {
        VoucherEntry entry = convertToEntry(entryDto);
        entry.setVoucherId(voucher.getId());
        voucherEntryMapper.insert(entry);
    }
    
    return success(voucher);
}
7.2.2 凭证记账（更新科目余额）
业务规则：
•	只有已审核的凭证才能记账
•	记账时更新科目余额表（本币发生额）
•	更新逻辑：根据凭证中的借方/贷方金额，累加到对应科目当前期间的 debit_amount 和 credit_amount 字段，并重新计算期末余额
伪代码：
text
@Transactional
public void bookkeep(Long voucherId) {
    Voucher voucher = voucherMapper.selectById(voucherId);
    if (voucher.getStatus() != 1) {
        throw new BusinessException("只有已审核的凭证才能记账");
    }
    
    List<VoucherEntry> entries = voucherEntryMapper.selectByVoucherId(voucherId);
    String period = voucher.getPeriod();
    
    for (VoucherEntry entry : entries) {
        // 获取或创建科目余额记录
        SubjectBalance balance = balanceMapper.selectBySubjectIdAndPeriod(
                entry.getSubjectId(), period);
        if (balance == null) {
            // 初始化余额（从上一期间结转期初）
            balance = initBalance(entry.getSubjectId(), period);
        }
        // 更新发生额
        balance.setDebitAmount(balance.getDebitAmount().add(entry.getDebitAmount()));
        balance.setCreditAmount(balance.getCreditAmount().add(entry.getCreditAmount()));
        // 重新计算期末余额
        recalcEndingBalance(balance);
        balanceMapper.updateById(balance);
    }
    
    voucher.setStatus(2);
    voucher.setBookkeeperId(currentUserId());
    voucherMapper.updateById(voucher);
}

    List<AccountSubject> assetSubjects = subjectMapper.selectByTypeAndLevel(1, 1);

        SubjectBalance balance = balanceMapper.selectBySubjectIdAndPeriod(
                subject.getId(), period);

        BigDecimal amount = balance.getEndingDebit();
        vo.addAssetItem(subject.getSubjectName(), amount);

    vo.setTotalAsset(totalAsset);
    
    // 负债合计

                subject.getId(), period);
        // 负债类科目取期末贷方余额
        BigDecimal amount = balance.getEndingCredit();
        vo.addLiabilityItem(subject.getSubjectName(), amount);
        totalLiability = totalLiability.add(amount);

    vo.setTotalLiability(totalLiability);

    // ...
    
    vo.setTotalEquity(totalEquity);
    vo.setTotalLiabilityAndEquity(totalLiability.add(totalEquity));
    
    return vo;
}
7.3 安全设计
7.3.1 Spring Security + JWT 配置
•	自定义 JwtAuthenticationFilter，解析请求头中的 Token，验证并设置 Authentication。
•	配置 SecurityConfig，放行登录、静态资源等公开接口，其余接口均需认证。
•	使用 @PreAuthorize 注解进行方法级权限控制。
关键代码示例：
java
@Configuration
@EnableWebSecurity
@EnableGlobalMethodSecurity(prePostEnabled = true)
public class SecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf().disable()
            .sessionManagement().sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            .and()
            .authorizeRequests()
            .antMatchers("/api/auth/**", "/swagger-ui/**").permitAll()
            .anyRequest().authenticated()
            .and()
            .addFilterBefore(jwtAuthenticationFilter(), UsernamePasswordAuthenticationFilter.class);
        return http.build();
    }
    
    @Bean
    public JwtAuthenticationFilter jwtAuthenticationFilter() {
        return new JwtAuthenticationFilter();
    }
}

// 请求拦截：添加 token
axios.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers['Authorization'] = 'Bearer ' + token;
  }
  return config;
});

// 响应拦截：统一处理错误
axios.interceptors.response.use(
  response => response.data,
  error => {
    if (error.response.status === 401) {
      // 未认证，跳转登录
      router.push('/login');
    }
    return Promise.reject(error);
  }
);
